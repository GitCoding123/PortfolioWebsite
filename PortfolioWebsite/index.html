About Me

I went to high school at Ponte Vedra High School (PVHS) and graduated before the summer of 2013. Once I obtained my Computer Science degree, I have been searching for a software development role in Jacksonville, Florida.

Skills



The first programming language I learned in school throughout my physics degree was Python. We used this language in my Mathematical Physics lab by utilizing it's vast amount of mathematical libraries. One example of this was using the library 'numpy' in order to graph and visually see a fourier series mapped out on a graph. This intrigued me to learn more about python and programming in general. I then started to have fun with it and created some mini games like tic-tac-toe and rock paper scissors. Eventually, I started to learn about 2-D platform games and took a stab at trying to create one. This lead me to pursue a degree in Computer Science where I learned much more about the concept of programming and a few different laguages, such as Java.

With Java, I was able to effectively understand the importance of object-oriented programming (OOP). OOP is a programming principle in computer science that relies on the concept of objects and classes. It is used to structure a software program into simple, reusable pieces of code blueprints, which are used to create individual instances of objects. Some of these languages are not necessarily restricted to the OOP paradigm, such as JavaScript, Python, and even PHP. They can allow for both procedural and OOP programming styles. Recently, I have been working on Udemy courses for JavaScript and React, expanding my horizon by delving into the realm of web development.
